# Design Minimal Skill

Скилл создаёт одну standalone HTML-страницу в сдержанном минимальном стиле: чтение на первом месте, сильные отступы, тонкие рамки, ограниченная палитра и никаких декоративных эффектов.

## Что это

Используй, когда нужен аккуратный спокойный экран: дашборд агента, бриф, раздатка, отчёт, операционная карта, чек-лист или внутренний вид процесса.

На выходе один `.html` файл с Tailwind CDN и системными шрифтами. Скилл избегает кастомного component CSS, JS-фреймворков, иконок, градиентов, теней, скруглений и responsive-вариантов.

## Когда применять

- Дашборды агента
- Раздатки для урока или воркшопа
- Внутренние брифы
- Карты процесса
- Плотные операционные экраны
- Отчёты, где важнее чтение и структура

Не подходит:

- Для маркетинговых лендингов
- Для богатых интерактивных приложений
- Для mobile-first страниц
- Для иллюстративных hero-экранов
- Для SVG-heavy диаграмм

## Контракт дизайна

- Один HTML-файл
- Только Tailwind CDN
- CSS-переменные только в `:root`
- Tailwind-классы прямо на элементах
- Фиксированный desktop-oriented layout
- Без `sm:`, `md:`, `lg:`, `xl:`, `2xl:` классов
- Без media queries
- Без внешних шрифтов
- Без библиотек иконок
- Без теней, градиентов, скруглений и ярких accent-палитр

## Правило языка

Видимый интерфейс пишется на языке пользователя. Для русских страниц заголовки, подписи, объяснения статусов и footer copy должны быть по-русски. Английский остаётся там, где он технический и должен копироваться без перевода: пути файлов, команды, code identifiers, schema fields, API names, названия моделей и literal values.

## Установка

```bash
cp -r skills/design-minimal ~/.claude/skills/
```

После этого скилл доступен в Claude Code.

## Как запустить

Скажи агенту:

> «Используй design-minimal и сделай standalone HTML dashboard для этого процесса».

Или:

> «Сделай минимальную HTML-раздатку для этого урока».

Агент создаст один HTML-файл по строгому визуальному контракту.
